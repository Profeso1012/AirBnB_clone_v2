# C is fun!

This project contains a Flask web application that listens on `0.0.0.0` at port `5000` and provides the following routes:
- `/`: Displays "Hello HBNB!"
- `/hbnb`: Displays "HBNB"
- `/c/<text>`: Displays "C " followed by the value of the `text` variable (underscores in `text`
